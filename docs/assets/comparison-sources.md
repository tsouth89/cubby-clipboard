# Comparison source notes

Checked July 18, 2026. The public graphic deliberately compares a small set of verifiable capabilities and includes a “where it shines” row so competing products receive credit for their strengths.

## Windows clipboard history

- Microsoft documents the Win+V shortcut, a 25-item limit, optional cross-device sync, and that unpinned history is cleared when the PC restarts: https://support.microsoft.com/en-us/windows/apps/using-the-clipboard

## Ditto

- Ditto documents its default Ctrl+backtick shortcut, persistent database, portable build, local-first behavior, broad clipboard-format support, and GPL-3.0 source: https://ditto-cp.sourceforge.io/ and https://github.com/sabrogden/Ditto
- The graphic says OCR is “Not advertised” rather than asserting that no plugin or script can provide it.

## Raycast

- Raycast documents Windows support, optional clipboard text recognition for images, retention controls, and that login is no longer required: https://manual.raycast.com/clipboard-history and https://www.raycast.com/changelog/windows
- Raycast is credited for integrating clipboard history into a broader productivity launcher.

## Cubby

- Cubby claims are grounded in this repository: SQLite persistence, AES-256-GCM payload encryption with a Windows DPAPI-protected key, Win+V replacement, local OCR search, folders, ignored applications, and GPL-3.0 licensing.
- “No desktop analytics” applies only to the installed Cubby app. Website analytics are separate and do not receive clipboard content.
