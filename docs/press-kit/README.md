# Cubby press kit

Cubby is a free, open-source clipboard history replacement built for Windows 11. This folder contains publication-ready video, GIF, still images, and logo files. Every clip and screenshot uses staged content; no personal clipboard data appears in these assets.

## Included files

- `cubby-ocr-demo-1080p.mp4` — 15-second H.264 product demo
- `cubby-ocr-demo.gif` — lightweight animated demo
- `01-ocr-search.png` — search-inside-screenshots hero
- `02-project-folders.png` — folder workflow
- `03-privacy-controls.png` — local privacy controls
- `cubby-mark.svg` — canonical scalable Cubby mark
- `cubby-mark-256.png` — canonical transparent PNG mark
- `cubby-app-icon.svg` / `cubby-app-icon-256.png` — compatibility copies of the same canonical mark
- `description.txt` — ready-to-use product description
- `links.txt` — website, download, source, and contact links

Please credit the product as **Cubby** or **Cubby Clipboard**.
