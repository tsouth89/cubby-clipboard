# Cubby Clipboard website

Static product and policy pages for [cubbyclipboard.com](https://cubbyclipboard.com).

## Pages

- `index.html` — product landing page
- `privacy.html` — current local-data and privacy behavior
- `support.html` — setup guidance, FAQ, and issue-reporting links
- `terms.html` — open-source software terms
- `styles.css` — shared responsive design system

The site is plain HTML and CSS with no build step or client-side dependencies. Preview it with any local static-file server rooted in this directory.

The primary domain is `cubbyclipboard.com`. `cubbyclip.com` can redirect to it when DNS and hosting are configured.
